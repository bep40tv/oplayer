## Start as New Project

### Install

<Tabs items={['npm', 'script']}>
  <Tab>
```shell
npm i @oplayer/core @oplayer/ui
```

```js filename="app.js"
import Player from '@oplayer/core'
import ui from '@oplayer/ui'

Player.make('#oplayer', {
  source: {
    src: 'https://cdn.jsdelivr.net/gh/shiyiya/QI-ABSL@master/o/君の名は.mp4',
    poster: 'https://cdn.jsdelivr.net/gh/shiyiya/QI-ABSL@master/o/poster.png'
  }
})
  .use([ui()])
  .create()
```

  </Tab>

  <Tab>
```html filename="app.html"
<script src="https://cdn.jsdelivr.net/npm/@oplayer/core@latest/dist/index.ui.js"></script>
<script>
  OPlayer.make('#oplayer', {
    source: {
      src: 'https://cdn.jsdelivr.net/gh/shiyiya/QI-ABSL@master/o/君の名は.mp4',
      poster: 'https://cdn.jsdelivr.net/gh/shiyiya/QI-ABSL@master/o/poster.png'
    }
  })
    .use([OPlayer.ui()])
    .create()
</script>
```

</Tab>
</Tabs>

<Callout>`@Oplayer/ui provides a basic set of skin styles.</Callout>
