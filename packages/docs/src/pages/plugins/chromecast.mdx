## Chromecast

```shell
npm i @oplayer/plugins
```

```js
import { chromecast } from '@oplayer/plugins'
```

```html
<!-- Chromecast -->
<script src="https://cdn.jsdelivr.net/npm/@oplayer/plugins@latest/dist/chromecast.min.js"></script>
```

### Chromecast

```js
Player.make('#oplayer').use([chromecast]).create()
```
